# Team Call Scheduling System

## Overview

A FastAPI backend application that schedules meetings based on participant availability.

## Features

- Team Member Management
- Availability Management
- Meeting Requests
- Meeting Participants
- Automatic Scheduling
- Conflict Detection
- Search
- Sorting
- Pagination
- Meeting Status Updates

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- MySQL
- Pydantic

## Installation

```bash
git clone <repository>

cd team_scheduler

pip install -r requirements.txt
```

Run

```bash
uvicorn app.main:app --reload
```

API Documentation

```
http://127.0.0.1:8000/docs
```
